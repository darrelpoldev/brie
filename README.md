# BRIE 🎈

A free, Montessori-inspired toddler play app for ages 2–4.

**Play, explore & discover** — no ads, no rules, no data collected. Just play!

## Toys

- **Shape Play** — Tap shapes to drop them on a canvas, then drag them around
- **Patterns** — Complete shape patterns by picking the right piece

## Getting started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tech

- Vite + React 18
- Web Audio API for sound feedback
- No external dependencies beyond React
- Mobile-first, touch-friendly
